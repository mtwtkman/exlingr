Summary
↑

Get archives from specified room. Access to room/get_archives with session, room, before, limit. * before means "max message ID"

Post to specified room. Access to room/say with session, room, nickname, text

Show room list. Access to room/show with session, room

Subscribe on. Access to room/subscribe with session, room, reset
unsubscribe(room)

Subscribe off. Access to room/unsubscribe with session, room
